Red Dog Labs is an indie app studio building useful, well-crafted tools. Home of [BillStack](https://reddoglabs.com/#apps) and more.

## About

This is the marketing site for Red Dog Labs — a static site deployed on Netlify.

## Stack

- Plain HTML, CSS, and vanilla JS — no frameworks, no build step
- Hosted on [Netlify](https://www.netlify.com)
- Security headers and caching configured via `netlify.toml`

## Structure

```
├── index.html       # Main landing page
├── privacy.html     # Privacy policy
├── terms.html       # Terms of service
├── style.css        # Styles
├── main.js          # Interactions (nav toggle, animations)
├── favicon.svg      # Site icon
└── netlify.toml     # Netlify config (headers, redirects)
```

## Development

Just open `index.html` in a browser — no build tools needed.
